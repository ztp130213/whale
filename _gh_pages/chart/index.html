标签云

标签云的实现

实例

参数

标签云插件初始化时可以配置一些参数，以对象的形式传给whaleTagsCloud()

参数名 类型 默认值 描述
width Integer 800 生成图宽度
height Integer 260 生成图高度
tspeed Integer 11 标签移动速度
tags array [] 标签数据
radiusX Integer 90 竖轴半径
radiusY Integer 135 横轴半径
radiusZ Integer 90 z轴半径
