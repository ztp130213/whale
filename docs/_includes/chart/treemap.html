treemap

基于d3实现的treemap

实例

{% highlight html %}
{% endhighlight %} {% highlight js %} $.get('/assets/json/flare.json') .then(function (data) { $('#treemap-demo').treemap({ data: data, width: 650, height: 470, onChangePath: function (path) { console.log(path) } }) }); {% endhighlight %}

参数

标签云插件初始化时可以配置一些参数，以对象的形式传给whaleTagsCloud()

参数名 类型 默认值 描述
width Integer 718 画布宽度
height Integer 600 画布高度
data Object {} 图数据，见demo
onChangePath function undefined 改变方法
